
조종사
	이름
	체력
	특성
		저격수 : 초탄 정확도 숙련도에 버프
		돌격병 : 반동 제어 숙련도에 버프
		무예 통달 : 근접 숙련도에 버프
		통증 둔감 : 방어 숙련도에 버프
		등등
	숙련도
		원거리
			반동 제어 = 연사 총기 명중률 보정
			초탄 정확도 = 단발 총기 명중률 보정
		근접
			한손 무기 숙련 = 한손 무기 명중률 보정
			양손 무기 숙련 = 양손 무기 명중률 보정
		방어
			방패 숙련 = 유닛 데미지 감소
			충격 회복 = 조종사 데미지 감소
			

유닛
	이름 = 파츠 조합식 or 입력
	내구도 = 파츠별 내구도 +
	중량 = 파츠별 중량 +
	체력 = 조종사의 체력

	파츠별
		머리
			이름
			내구도
			중량
			필요 파워
		몸통
			이름
			내구도
			중량
			* 파워 출력 = 마석 등급 * 회로 구성 버프
			코어 : 등급에 따라 넣을 수 있는 개수 증가
				회로 구성 = 내구도 및 출력 버프
					순환 lv 1, 2, 3 = 출력 버프
					방어막 lv 1, 2, 3 = 내구도 버프
					경량 lv 1, 2, 3 = 이동 거리 버프
					저항 lv 1, 2, 3 = 속성 저항 버프
					마탄 lv 1, 2, 3 = 속성 데미지 버프
					등등 => 세트 회로 버프. 조립을 완성하고 나서 유닛 속성에서 표시. 조립,교체에 있어서 비용을 지불하여 세트 버프에 대하여 비용이 발생하도록.
							ex) 순환 1+2+3 : 출력&이동 버프. 내구도&속성저항 디버프.
								방어막 1+2+3 = 내구포&저항 버프. 출력&이동 디버프.
				마석 등급 = 기본 출력 값

			특징
				종류
					근접

							길이&크기 = 특징에 따른 피해량 배율. 최대 1.5 최소 0.5.
							재질 = 기본 피해량
						손잡이
							길이 = 특징에 따른 명중률 배율. 최대 1.5 최소 0.5.
							재질 = 기본 명중률
						특징
							도끼 = 길이에 대한 최소 최대치.
							검 = 길이에 대한 최소 최대치.
							창 = 길이에 대한 최소 최대치.
					원거리
						재질 : 총열 길이 및 구경에 대한 제약
						총열 길이 = 명중률 보정
						총열 구경(& 약실 크기) = 호환 탼약 결정 및 피해량 보정.
							대 : 일반, 고폭

						장전 방식

									고폭 : 화염, 부식, 냉기, 전기 특성 추가. 일반피해에 패널티
									관통 : 장갑관통 추가
										확률 : 치명타 확률
										패해 : 치명타 피해 배율
									일반 : 기본 피해량
							공격력 = 기본 피해량
							집탄율

데미지
	일반 = 공격력 * 버프 + 화명 * (화염 데미지에 대한)버프 + 부식 * (부식 데미지에 대한)버프
	치명타 = 공격력 * 버프 * 관통 피해
	체력 데미지 = 백어택인 경우에 받는 데미지의 일정 비율 or 최대 내구도의 10%를 넘는 데미지가 한턴에 들어오는 경우 등
명즁 = (무기 종류에 대한) 조종사 숙련도 * 무기 집탄율 * ( 무기의 사정거리 / 거리 ) * 버프
치명타 확률 = 장갑관통 * 100 초과 명중 * 버프

	장갑관통 : 크리티컬 확률
	화염 : 턴 당 데미지
	부식 : 장갑에 추가 데미지
	냉기 : 행동력 감소 => 디버프
	전기 : 명중률 감소 => 디버프

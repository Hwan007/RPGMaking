
조종사
	이름
	체력
	특성
		저격수 : 초탄 정확도 숙련도에 버프
		돌격병 : 반동 제어 숙련도에 버프
		무예 통달 : 근접 숙련도에 버프
		통증 둔감 : 방어 숙련도에 버프
	숙련도
		원거리
			반동 제어 = 연사
			초탄 정확도 = 단발
			장전 기술
		근접
			한손 무기 숙련
			양손 무기 숙련
		방어
			방패 숙련
			충격 회복
			

유닛
	이름 = 파츠 조합식 or 입력
	내구도 = 파츠별 내구도 +
	중량 = 파츠별 중량 +
	체력 = 조종사의 체력

	파츠별
		머리
			이름
			내구도
			중량
			필요 파워
		몸통
			이름
			내구도
			중량
			* 파워 출력
		팔
			이름
			내구도
			중량
			필요 파워
		다리
			이름
			내구도
			중량
			필요 파워
		무기
			이름
			내구도
			중량 : 명중률에 패널티
				근접 = 날 및 손잡이 길이 & 크기
				원거리 = 총열 길이 + 총열 구경
			특징
				종류
					근접
						날
							길이&크기 = 특징에 따른 피해량 배율. 최대 1.5 최소 0.5.
							재질 = 기본 피해량
						손잡이
							길이 = 특징에 따른 명중률 배율. 최대 1.5 최소 0.5.
							재질 = 기본 명중률
						특징
							도끼 = 길이에 대한 최소 최대치.
							검 = 길이에 대한 최소 최대치.
							창 = 길이에 대한 최소 최대치.
					원거리
						재질 : 총열 길이 및 구경에 대한 제약
						총열 길이 = 명중률 보정
						총열 구경(& 약실 크기) = 호환 탼약 결정 및 피해량 보정.
							대 : 일반, 고폭
							중 : 일반, 관통
							소 : 일반
						장전 방식
							자동 : 1회 공격에 연발 사격
							수동 : 정확도 보정
			* 공격력
				종류
					근접
						
					원거리
						에너지
							* 필요 파워
							공격력
							집탄율
						물리
							* 탄약 종류
								장탄수
								무게
								보유 탄약
								특징
									고폭 : 화염, 부식, 냉기, 전기 특성 추가. 일반피해에 패널티
									관통 : 장갑관통 추가
										확률 : 치명타 확률
										패해 : 치명타 피해 배율
									일반 : 기본 피해량
							공격력 = 기본 피해량
							집탄율

데미지
	일반 = 공격력 + 화명 + 부식
	치명타 = 공격력 * 관통 피해
명즁 = (무기 종류에 대한) 조종사 숙련도 * 무기 집탄율 * ( 무기의 사정거리 / 거리 )
치명타 확률 = 장갑관통 * 100 초과 명중

	장갑관통 : 크리티컬 확률
	화염 : 턴 당 데미지
	부식 : 장갑에 추가 데미지
	냉기 : 행동력 감소 => 디버프
	전기 : 명중률 감소 => 디버프

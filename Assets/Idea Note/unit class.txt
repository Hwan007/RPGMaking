[데미지]
	기본 공식 = 공격력 * 버프/디버프 + 화염 * (화염 데미지에 대한)버프/디버프 + 충격 * (충격 데미지에 대한)버프/디버프
	치명타 공식 = 공격력 * 버프 * 관통 피해
	
	기체 체력 = 기체 체력 - (기본 공식)
	방패 체력 = 방패 체력 - (공격력 * 버프/디버프 + 화염 * (화염 데미지에 대한)버프/디버프 + 충격 * (충격 데미지에 대한)버프/디버프 * 2배)

[명중률]
	기본 공식 = (무기 종류에 대한) 조종사 숙련도 * 무기 집탄율(최대 1 최소 0) * ( 무기의 사정거리 / 거리 ) * 버프/디버프
	치명타 확률 = 100 초과 명중 * 버프/디버프

					원거리	:	충전된 마나를 사용하여 일정 시간 동안 낮은 온도로 유지하여, 접촉 상대방에게 동상을 유발한다.
